## 5. Inheritance

a way for one class to extend another class.

```javascript
class Shape {
  constructor(width, height, color) {
    this.width = width;
    this.height = height;
    this.color = color;
  }

    console.log(`drawing ${this.color} color!`);
  }

  getArea() {
    return this.width * this.height;
  }
}

class Rectangle extends Shape {}

  draw() {
    super.draw();
    console.log('ğŸ”º');
```
